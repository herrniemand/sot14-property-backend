LANDBANKERS R US.

For Summer of Tech 2014, the Landbankers came together to find out how we could make sensible assessments based off visualised data. The project evolved into the creation of a system that would layer D3 visualisations on top of a MapBox frame. We aim to be able to answer questions using these visualisations. Questions such as:

  • How have property values changed over time, in a given area?
  • How has a given suburb gentrified over time?
  • What company owns the land in a given area?

Clear Representation of Discrete Data

If you can't make sense of the data you have, you may as well not have it.

Simple Systems

We designed this system with the aim to minimise moving parts, and therefore points of failure.

Gears before Beers

We aimed to satisfy a use case at the base level, before expanding application functionality beyond our initial plans.
